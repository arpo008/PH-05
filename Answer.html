What is the Document Object Model(DOM)

Ans: The Document Object Model (DOM) is a programming interface that represents the structure of an HTML or XML
document as a tree of objects. It allows scripts to access, manipulate, and modify the content,
structure, and style of a web page dynamically.

How do you select an element from the DOM

Ans:To select an element from the DOM, you use JavaScript methods :
- getElementById(id): Selects an element by its unique `id`
- getElementsByClassName(class): Selects all elements with a specific class.
- querySelector(selector): Selects the first element that matches a CSS selector.
- querySelectorAll(selector): Selects all elements that match a CSS selector.

What is event delegation in the context of the dom ,and why is it useful?

Ans:Event delegation is a technique in the DOM where a single event listener
is attached to a parent element to handle events for multiple child
elements, instead of adding listeners to each child.
It's useful for optimizing performance, especially when dealing with dynamically added elements.

How do you manipulate an elements attributes and styles using the dom?

Ans:To manipulate an element's attributes using the DOM, you can modify its properties like `id`, `class`, or `src` directly
or through methods that set or get attributes. For styling, you can change the element
's appearance by adjusting its `style` property,
allowing you to update CSS properties like color, background, or
size dynamically.
